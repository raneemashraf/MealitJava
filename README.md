## Mealit is an Android Mobile Application that helps the user plan meals. and view categories, suggesting meals, and searching for a specific meal using many options.

# Key Features:

### -Meal of the day
### -Search by Name of Meal
### -Filter by (area, ingredients, categories) 
### -Save Favorite and Planner 
### -Multi-account support 
### -Offline access to Favorite and Planned meals 
### - Backup and Synchronization for favorites and Planner using Firebase 
### -Guest mode with search and view meals.


# Technologies Used: 

### -MVP Architecture Pattern 
### -Retrofit 
### -RxJava 
### -Room 
### -Firebase Authentication 
### -Firebase Firestore 
### -Material Design 
### -Singleton Pattern 
### -Repository Pattern 
### -Glide for image loading 
### -Lottie files for animation 
### -Navigation component
